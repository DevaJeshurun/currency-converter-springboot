# Currency Converter - Spring Boot Project

A web-based currency converter built using Spring Boot and Thymeleaf.

## Features
- Live exchange rates using external API
- Searchable dropdown UI
- Currency swap option

## Tech Stack
- Java
- Spring Boot
- Thymeleaf
- REST API

## Run Locally
1. Clone repo
2. Run CurrencyApplication.java
3. Open http://localhost:8080/
