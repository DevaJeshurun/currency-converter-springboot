# Currency Converter App

A fast and reliable currency converter application that allows users to convert between different currencies using real-time exchange rates fetched from an external API.

This application provides accurate currency conversion with a simple and user-friendly interface, making it easy for users to check exchange rates instantly.

---

## Features

### Real-Time Currency Conversion
- Convert between multiple international currencies
- Fetch live exchange rates from API
- Accurate and up-to-date conversion results

### User-Friendly Interface
- Simple and clean UI
- Easy currency selection
- Instant conversion results

### Multiple Currency Support
- Supports major global currencies such as:
  - USD (US Dollar)
  - INR (Indian Rupee)
  - EUR (Euro)
  - GBP (British Pound)
  - JPY (Japanese Yen)
  - and more

### Fast Performance
- Lightweight application
- Quick API response handling
- Smooth user experience

## How It Works

1. User selects base currency
2. User selects target currency
3. User enters amount
4. App fetches exchange rate from API
5. App calculates and displays converted amount
